# Student_grading_system
I developed a Student Grade Management System using Streamlit and Pandas.
It is a web-based application that allows users to add, view, search, update, and delete student records.
The system automatically calculates total marks, average, and grade classification.
I also implemented a dashboard for analytics, showing overall performance like average marks, grade distribution, and topper details.
Data is stored using a CSV file, making it lightweight and easy to manage
